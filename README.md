## Description
The educational task is to create a person’s identity card. Data must include avatar, name, title, telephone number and email address.
***

## Elements
  
* AppBar + Text Widget
* Circular Avatar + Image
* Divider
* Scaffold
* Button Widget + exit
* Text Widget
* SizedBox
